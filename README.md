# NextJS + Redux

NextJS app setting with Redux

## Features

- Redux (with @reduxjs/toolkit)
- Husky + lint-staged + commitlint (follow commit-conventional)
- Absolute imports

## Running

```bash
# For development
yarn run dev

# For production
yarn run build && yarn run start
```
